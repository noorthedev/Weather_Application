🌤 Weather App

A modern and responsive Weather Application built using HTML, CSS, and JavaScript. The app allows users to search weather information for any city in real time using the OpenWeatherMap API.

✨ Features:
• Real-time weather data fetching
• City-based weather search
• Temperature, humidity, and wind speed display
• Feels-like temperature information
• Weather condition icons and descriptions
• Error handling for invalid city names
• Responsive and clean glassmorphism UI
• Enter key support for quick search

🛠️ Technologies Used:
• HTML5
• CSS3
• JavaScript (ES6)
• OpenWeatherMap API

This project demonstrates API integration, asynchronous JavaScript (Fetch API), DOM manipulation, and responsive UI design.
